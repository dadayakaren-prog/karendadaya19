Installing and Comparing on Operating System in Virtual Box

Step-by-Step Process:

Step 1: Prepare Your Tools ● Ensure VirtualBox is installed. ● Download a compatible ISO file of an operating system (e.g., ubuntu-22.04.iso or Windows10.iso)

.Screenshot 1: Show your downloaded ISO file and VirtualBox installed on your machine.

Step 2: Create a Virtual Machine

1. Open VirtualBox and click "New". 2. Name your VM (e.g., “Ubuntu VM”). 3. Select the appropriate Type and Version (e.g., Type: Linux, Version: Ubuntu 64-bit). 4. Click Next.

Screenshot 2: Capture the "Create Virtual Machine" window showing name, type, and version settings.

Step 3: Configure the Virtual Machine

1. Set the memory size (RAM): Minimum 2 GB for Linux, 4 GB for Windows. 2. Choose “Create a virtual hard disk now”, then click Create. 3. Select VDI → Dynamically Allocated. 4. Set a disk size (at least 20 GB for Linux, 50 GB for Windows). 5. Click Create.

Screenshot 3: Show your memory and hard disk size configuration screens.

Step 4: Attach the ISO File

1. Select your VM → Click Settings → Go to Storage. 2. Under “Controller: IDE”, click Empty, then on the right side, click the disk icon. 3. Choose “Choose a disk file” → Select your ISO file → Click OK.

Screenshot 4: Show the ISO file successfully loaded in the Storage section.

Step 5: Start the VM and Install the OS

1. Click Start to run your VM. 2. Follow the OS installation process: o Choose language, region, and keyboard layout. o Set up user account and password. o Complete installation.

Screenshot 4: Show the ISO file successfully loaded in the Storage section.
